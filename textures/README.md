# Texture Files Required

Place the following high-resolution texture files in this directory:

1. **8k_earth_daymap.jpg** 
   - Earth surface texture (day view)
   - Resolution: 8192 x 4096 or higher
   
2. **8k_earth_clouds.jpg**
   - Cloud layer texture
   - Resolution: 8192 x 4096 or higher

3. **8k_stars_milky_way.jpg**
   - Milky Way skybox texture (cosmic background)
   - Resolution: 8192 x 4096 or higher

## Download Sources

### Option 1: Solar System Scope (Recommended)
https://www.solarsystemscope.com/textures/

1. Navigate to the website
2. Download "Earth Daymap" (8K)
3. Download "Earth Clouds" (8K)
4. Rename files if needed to match the names above

### Option 2: NASA Visible Earth
https://visibleearth.nasa.gov/

Look for:
- "Blue Marble" series for daymap
- Cloud texture compilations

## File Naming
Make sure the files are named EXACTLY as:
- `8k_earth_daymap.jpg`
- `8k_earth_clouds.jpg`
- `8k_stars_milky_way.jpg`

Otherwise, update the texture paths in `Earth.jsx` and `BackgroundGalaxy.jsx`
